# TUI Finder Tool *rednif*

## WHAT

コマンド入力ではなく、TUI で視覚的、直感的にディレクトリ、ファイル操作をおこなうことが出来るツール。

`rednif` ではディレクトリ、ファイルに対して以下の操作をおこなうことが出来る。

- 開く(Open)
- コピー(Copy)
- 複製(Clone)
- 名前変更(Rename)
- 削除(Delete)
